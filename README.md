# 🚗 Isiqalo Car Wash

Professional car wash website for **Dube, Soweto**. Service menu, pricing, hours, location, and bookings over WhatsApp.

🔗 Live: _(link once deployed)_

---

## Stack

Static site — HTML / CSS / vanilla JavaScript. Designed to load fast on low-data township connections and run from any host (Netlify, GitHub Pages, shared hosting).

---

## Features

- Service and pricing grid
- Promo banner (editable without a rebuild)
- WhatsApp "Book Now" CTA
- Google Maps embed for location
- Mobile-first responsive

---

## Local development

```bash
git clone https://github.com/ChumaMike/isiqalo-car-wash.git
cd isiqalo-car-wash
# Open index.html in a browser, or serve:
python -m http.server 8000
```

---

## Deployment

Drag-and-drop to Netlify, or push to any static host. No build step required.
